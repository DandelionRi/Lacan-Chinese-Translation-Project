# 拉康中文开放翻译计划

拉康中文开放翻译计划是一个面向中文读者的开放翻译项目，旨在整理、翻译、校订和维护雅克·拉康研讨班及相关文本的中文材料。


## 阅读说明

- “原文”开关用于显示或隐藏法文原文。
- “注释”开关用于显示或隐藏译注、术语说明和文本校订提示。
- “个人解读评论”开关用于显示或隐藏译者的阅读札记和解释性评论。

## 协作说明

长期维护内容放在 `texts/` 目录中；`src/` 和 `book/` 都是构建生成结果，不作为源文件提交。

原文和译文通过分段 ID 对齐，分段 ID 通常类似 `s8-01-0001`。在评论、校验、交流和 Pull Request 审阅过程中，请尽量用分段 ID 指明具体段落，方便参与者定位同一处文本并追踪后续修订。
